# AI 建工法律咨询 GEO 营销系统 (AI Construction Law Geo Marketing System)

## 项目简介 (Project Overview)

本项目是一个专为建筑工程法律咨询领域设计的智能营销内容生成系统。它利用先进的生成式 AI 技术（Google Gemini 3 模型），不仅能通过上传的案件录音整理稿或案情文本自动提取客户痛点、分析证据链，还能一键生成高质量的营销内容，包括微信公众号文章、短视频脚本和知乎专业回答。

系统的核心目标是帮助律所和律师团队通过技术手段提升案源转化率，将复杂的法律专业知识转化为引人入胜的营销素材，实现“GEO”（可能是指地理位置相关的精准营销或通用的 Global/General Marketing 策略）营销目标。

## 核心功能 (Core Features)

1.  **智能案情分析 (AI Analysis)**
    *   **文件上传**: 支持 Word (.docx) 和文本 (.txt) 格式的案情/录音整理稿上传。
    *   **痛点提取**: 自动识别案件中的核心法律纠纷点、客户痛点和风险点。
    *   **证据分析**: 梳理案件的关键证据和法律逻辑。

2.  **多渠道内容生成 (Multi-channel Content Generation)**
    *   **公众号文章**: 根据案情自动生成深度专业且具吸引力的普法文章。
    *   **短视频脚本**: 生成适合抖音/视频号的口播文案和分镜脚本。
    *   **知乎回答**: 生成符合知乎社区氛围的专业法律解答。
    *   **自定义营销方向**: 支持注入特定的营销导向（如“北京建工法律纠纷解决方案”）。

3.  **模板与知识库管理 (Templates & Knowledge Base)**
    *   **案例库/模板库**: 自动将分析过的案例保存为模板，方便复用和沉淀。
    *   **律所知识库**: 集成律所专业知识库（`FIRM_KNOWLEDGE_BASE`），确保生成内容符合律所专业标准。

4.  **仪表盘 (Dashboard)**
    *   直观展示最近的分析记录、状态（处理中/完成/失败）和关键信息。

## 技术栈 (Tech Stack)

*   **前端框架**: React 18, TypeScript, Vite
*   **UI 组件**: Tailwind CSS (推测), Lucide React (图标)
*   **AI 模型**: Google Gemini 3 (通过自定义代理服务调用，支持 `gemini-3-pro-high`, `gemini-3-flash` 等模型)
*   **文档处理**: Mammoth.js (用于 .docx 解析)

## 快速开始 (Getting Started)

### 1. 环境准备
确保已安装 [Node.js](https://nodejs.org/)。

### 2. 安装依赖
```bash
npm install
```

### 3. 配置环境
在项目根目录创建或编辑 `.env.local` 文件，填入你的 Gemini API Key：
```env
VITE_GEMINI_API_KEY=your_api_key_here
# 可选：配置自定义代理地址
# VITE_ANTIGRAVITY_BASE_URL=http://127.0.0.1:8045
```

### 4. 启动开发服务器
```bash
npm run dev
```
访问终端输出的本地地址（通常是 `http://localhost:5173`）。

## 使用流程 (Workflow)

1.  **上传**: 在首页上传脱敏后的案子录音整理稿或案情描述文档。
2.  **分析**: 系统自动调用 AI 进行深度分析，稍等片刻即可看到提取的“痛点”和“案情摘要”。
3.  **生成**: 点击“生成营销内容”，选择生成的类型（文章/视频/问答）。
4.  **优化**: 在生成界面微调 AI 的输出，或保存到本地。
